Float and clear

I use a class to let it float left and it does, with all the text on the right even the the image is in the middle of the pargraph. I specified two styles (one for sizing and formatting the image, the other for specifying float) and used a compound class to apply them both to this image.

    .smallerImage
    {
      width:25%;
      border:1px green solid;
      margin:1em;
    }

    .leftFloat
    {
      float:left;
    }

    In the HTML: class="leftFloat smallerImage"

But I assigned a class with clear:left; to this one so the wrapping has stopped. More typically I'd apply the class to a div rather than a single paragraph.

    .stopLeftFloat

Display to change block level and inline

This image is set to display as a block level image using the Display property even though and image is an inline element. Notice the text is above and below and not left or right.with this paragraph because of the code I used to center images generally.

  .makeBlock
  {
    display:block;
    margin-left: auto;
    margin-right: auto;
  }

Display tables

There are a variety of Displayformats. Three of importance to us are

  • table: Assumes the associated div is going to hold rows or columns and table cells.
  • table-row: Acts like the trtag and assumes the associate div will hold cells of a table. There is a table-column style as well.
  • table-cell: Is used to indicate the associate div will act like a table cell, similer to the td tag.

In my CSS file I will create classes that assing these styles to the display property. I could choose to also add some formatting and handle things associate with the box model.

  .divTable
  {
    display:table;
    margin:1em;
    border:2px black solid;
  }

  .divRow
  {
    display:table-row;
  }

  .divCell
  {
    display:table-cell;
    width:200px;
    padding:1em;
    border:1px black solid;
  }

I can then make a table entirely out of Divs. Why? Because modern elements in divs are more flexible and accesible by Javascript. A table is still great for an actual table showing a tableau of data, but flexibility in format and access is better with "display tables".
